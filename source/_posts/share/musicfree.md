# MusicFree 免费音乐下载-开源音源接口汇总

## 软件下载

- 官方下载：[https://musicfree.catcat.work/](https://musicfree.catcat.work/)

- Github 项目地址：[GitHub - maotoumao/MusicFree: 插件化、定制化、无广告的免费音乐播放器](https://github.com/maotoumao/MusicFree)

## 一、软件介绍

- MusicFree 是一款开源、免费、可定制化的本地音乐播放器。

## 二、配置方法

## MusicFree 安卓版 配置方法

- 点击左上角按钮
- 点击【插件设置】
- 点击右下角加号
- 点击【从网络安装插件】
- 输入插件 URL 即可（插件链接以 .json 或 .js 结尾）
  ![Android](https://www.tao-space.top/2024/03/29/MusicFree%E9%9F%B3%E6%BA%90%E6%8E%A5%E5%8F%A3%E6%B1%87%E6%80%BB/android.png)

## MusicFree PC 版 配置方法

- 点击主界面左侧【插件管理】
- 点击顶部【从网络安装】
- 输入插件 URL 即可（插件链接以 .json 或 .js 结尾）
  ![pc](https://www.tao-space.top/2024/03/29/MusicFree%E9%9F%B3%E6%BA%90%E6%8E%A5%E5%8F%A3%E6%B1%87%E6%80%BB/pc.png)
